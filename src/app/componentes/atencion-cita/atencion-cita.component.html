¿Cómo funciona este módulo?
1. Procedimientos y Materiales
  1. Aquí debes agregar cada tratamiento realizado.
  2. Se debe ingresar el material principal usado.
  3. En la CANTIDAD: no es la cantidad de material, sino las veces que se aplicará el tratamiento.
  4. Si usaste materiales extras (guantes, gasas, etc.), agrégalos en la sección de "Insumos Sueltos".
Importante: Esto actualizará automáticamente el stock de insumos en el consultorio.
2. Nota Clínica

Redacta la evolución del paciente.
Es obligatorio registrar alergias o antecedentes si es la primera vez, y llenar el campo de "Avance del Tratamiento" para poder guardar. Para proximas citas no es requerido actualizar unicamente si el paciente presenta alguna nueva alergía.
