Guía Rápida: Agenda de Citas
1. ¿Cómo buscar citas?

Utiliza la barra superior para filtrar el listado. Puedes combinar ambos filtros:

  • Fecha: Selecciona un día específico del calendario.
  • Estado: Muestra solo citas pendientes, atendidas, canceladas, etc.
  • Botón Filtrar: Es necesario presionarlo para refrescar la tabla.
2. Estados de la Cita (Colores)

Cada color en la columna "Estado" indica una situación diferente:

Agendada
Cita programada, paciente esperado.
Confirmada
El paciente confirmó asistencia.
Atendida
Consulta finalizada con éxito.
Pendiente / Reprogramada
Requiere atención administrativa.
Cancelada / No Asistió
Cita perdida.
3. Acciones Disponibles
  • Agendada Abre el expediente clínico para registrar tratamientos y notas. Solo disponible si la cita está "Agendada" es decir que el paciente ya confirmo la cita.
  • Permite consultar lo que se le hizo al paciente en una cita ya finalizada es decir que la cita ya fue atendida.
